**HOMEWORK 3 NEURALNET README FILE**

The NeuralNet training program and AIbot are contained within the neuralNet.py file.
When initiated, it will read the data in the 'train.csv' file, separate it into
a training set and a dev set (an unadulterated version of the original set to test
training accuracy). The training program will output the accuracy every 10 iterations.
When called from the 'main.py' file, the training program will run, and then 4
test predictions are run. The AIbot will use the results of the training to analyze
4 separate selected images, and output the results to the user.
The main file will then make predictions using the dev set and output the accuracy.
